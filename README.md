# Emotion Detector AI Application

## Project Description
This is a final project for the IBM AI Development course. The application is a web-based tool developed using **Python**, **Flask**, and the **Watson NLP library**. It analyzes the emotion of a given text and returns scores for 'anger', 'disgust', 'fear', 'joy', and 'sadness', along with the 'dominant_emotion'.

## Features
- AI-powered emotion detection using IBM Watson NLP.
- Web interface built with Flask.
- Error handling for invalid or blank inputs.
- Static code analysis compliant with Pylint.
- Unit testing for core logic.

## How to Run
1. Clone the repository.
2. Install requirements (Flask, Requests).
3. Run `python3 server.py`.
4. Access the app via `localhost:5000`.
